# Investment Rules - ThesisVenture

## Buy Rules (ALL must be met)
- Ben: PASS (Moat verified, ROE>15%, below intrinsic value)
- Sarah: PASS or NEUTRAL (not parabolic)
- Alex: PASS or NEUTRAL (no governance failure)
If Ben NEUTRAL/FAIL or any FAIL → Do NOT Buy

## Hold Rules
- Hold while Economic Moat intact
- Price drops = opportunity, not exit signal
- Review every Quarter after earnings

## Sell Rules
- Ben FAIL: Moat permanently eroded
- Alex FAIL: Fraud or governance failure confirmed
- Extreme overvaluation beyond realistic FCF

## Position Sizing
- Initial: 10% ($1,000)
- Max per stock: 25%
- Scale in: max 3 tranches, 10% price apart
- Trim if position exceeds 25%

## Risk Management
- No leverage, no margin, no shorts
- Min 10% cash ($1,000) always
- No new BUY if cash < $1,000

## Fund Rules
- Capital: $10,000
- Max 10 stocks

## Portfolio Math (STRICT)
- Cash changes ONLY on BUY/SELL — never on price movement
- BUY: Cash = Cash - (Shares x Price)
- SELL: Cash = Cash + (Shares x Price)
- Total Value = Cash + Sum(Shares x Current Price)
- Return = (Total Value - $10,000) / $10,000 x 100%
